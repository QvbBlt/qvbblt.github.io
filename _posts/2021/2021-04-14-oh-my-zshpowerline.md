## oh my zsh

复制粘贴到终端运行。

`sh -c "$(wget -O- https://gitee.com/mcornella/ohmyzsh/raw/master/tools/install.sh)"`

将终端字体设置成powerline，防格式乱码。

## powerline

```
# clone
git clone https://github.com.cnpmjs.org/powerline/fonts.git
# install
cd fonts
./install.sh
```
